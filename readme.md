## Description
**Photo Blog** Allows you create a blog of only photos.

## Installation

Add it to your Gemfile:

```ruby
gem 'carrierwave_direct'
gem 'sidekiq'
gem 'rmagick'
gem 'carrierwave'
gem 'fog'
```
Run the following command to install it:

```console
bundle install
rake db:migrate
```
